# 🚀 Инструкция по загрузке на GitHub

## Шаг 1: Создайте репозиторий на GitHub

1. Перейдите на https://github.com/new
2. Название репозитория: `campus-navigator-bot` (или любое другое)
3. Описание: `Telegram бот-навигатор для ПГНИУ с системой рейтинга`
4. Выберите **Public** или **Private**
5. **НЕ** создавайте README, .gitignore или лицензию (они уже есть)
6. Нажмите **Create repository**

## Шаг 2: Подключите удаленный репозиторий

После создания репозитория GitHub покажет команды. Выполните:

```bash
git remote add origin https://github.com/YOUR_USERNAME/campus-navigator-bot.git
git branch -M main
git push -u origin main
```

Замените `YOUR_USERNAME` на ваше имя пользователя GitHub.

## Шаг 3: Готово! 🎉

Ваш проект теперь на GitHub!

## 📝 Важно

Перед загрузкой убедитесь что:
- ✅ Удалены все `console.log` и отладочные `print`
- ✅ Нет секретных ключей в коде (используйте `.env`)
- ✅ Файл `.gitignore` настроен правильно

## 🔐 Безопасность

**НИКОГДА** не загружайте на GitHub:
- Токены ботов
- API ключи Supabase
- Пароли и секретные ключи

Используйте переменные окружения (`.env` файл) для хранения секретов.
